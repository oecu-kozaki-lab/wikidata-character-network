# Wikidata 人物関係抽出・可視化システム

Wikidata に蓄積された作品の登場人物データを活用し、人物間の複雑な関係性をネットワーク図として可視化する Web アプリケーションです。

## 概要

本システムは、大阪電気通信大学 情報通信工学部 情報工学科の卒業研究として開発されました。

- **研究テーマ**: Wikidata を用いた人物関係の抽出・可視化システムの開発と評価
- **著者**: GP22A018 榎本裕太
- **指導教員**: 古崎晃司

## 主な機能

- **作品検索**: 作品名を入力するだけで Wikidata から候補を検索・選択
- **人物関係の自動抽出**: SPARQL クエリにより登場人物と関係性データを自動取得
- **ネットワーク可視化**: Cytoscape.js による対話的なネットワーク図の描画
- **関係カテゴリの色分け**: 家族関係・社会的関係・知的関係・職業的関係を自動分類
- **CSV エクスポート**: 抽出データの外部ツール連携
- **グラフ履歴**: 過去の可視化結果を保存・復元
- **プリセット機能**: よく使う関係カテゴリの組み合わせを選択可能

## 使用技術

| 項目 | 技術 |
|---|---|
| データソース | Wikidata SPARQL API |
| 可視化ライブラリ | Cytoscape.js |
| フロントエンド | HTML / CSS / JavaScript (SPA) |
| データ保存 | LocalStorage |

## 使い方

1. `人物関係抽出システム.html` をブラウザで開く
2. 検索ボックスに作品名（の一部）を入力
3. 候補リストから作品を選択
4. 自動的に登場人物と関係性が抽出され、ネットワーク図が描画される

> **注意**: インターネット接続が必要です（Wikidata API へのアクセスのため）
